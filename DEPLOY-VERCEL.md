# Vercel root directory

If this repository is connected to Vercel (e.g. **github.com/samikciku/hackthetrash-2**), set:

**Root Directory:** `hackthetrash/frontend`

Do **not** use `frontend` alone — that folder only exists inside `hackthetrash/`.

See `hackthetrash/docs/VERCEL.md` for the full checklist.
