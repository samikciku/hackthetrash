# UI Mockups

Open `index.html` in a browser to see all mockup screens:

1. **Mobile Submission Form** - photo, GPS, tags, severity
2. **Public Map** - Desktop view with sidebar filters and color-coded markers
3. **Authority Dashboard** - stats cards + reports queue
4. **Landing Page** - hero, steps, CTAs

Files:
- `index.html` - all screens
- `styles.css` - design tokens and component styling

To convert to Figma later: import `index.html` via the html2figma plugin or rebuild with the design tokens listed in `styles.css` (see `:root` variables).
